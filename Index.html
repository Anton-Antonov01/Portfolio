Проекты

Веб приложение ИС Отель, написанное на ASP.NET Core, позволяет пользователю совершать онлайн бронирования комнат, смотреть список своих бронирований и проверять их статус, а администратору обрабатывать новые бронирования и управлять данными (аккаунтами, комнатами, категориями, бронированиями). В проекте использовалась трёхуровненвая архитектура состоящая из уровня бизнес логики, уровня работы с данными и уровня презентации. Базой данных для приложения был выбран MSSQL Server. Для работы с базой данных используется ORM Entity Framework. Уровень представления - MVC ASP.NET Core. В решении присутвует WEB API. Для авторизации, аутентификации и разбиения аккаунтов на роли используется Core Identity.
